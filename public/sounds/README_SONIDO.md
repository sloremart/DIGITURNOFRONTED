# Sonido de Notificación

Para que funcione el sistema de sonido en la pantalla de sala de espera, necesitas crear un archivo de audio.

## Opciones recomendadas:

### 1. Archivo MP3 (recomendado)
- Crea un archivo llamado `notification.mp3`
- Duración: 2-3 segundos
- Formato: MP3, 128kbps o superior
- Contenido: Sonido de campana, beep o notificación

### 2. Archivo WAV (alternativa)
- Crea un archivo llamado `notification.wav`
- Duración: 2-3 segundos
- Formato: WAV, 44.1kHz, 16-bit
- Contenido: Sonido de campana, beep o notificación

## Dónde colocar:
Coloca el archivo de audio en esta carpeta: `digiturnofrontend/public/sounds/`

## Ejemplos de sonidos:
- Campana de notificación
- Beep electrónico
- Sonido de "ding"
- Melodía corta de alerta

## Nota:
El sistema automáticamente detectará y reproducirá el sonido cuando haya nuevos turnos llamados.
